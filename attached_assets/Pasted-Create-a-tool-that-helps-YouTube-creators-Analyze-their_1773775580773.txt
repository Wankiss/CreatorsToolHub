⚙️ ANALYSIS LOGIC
1. Preprocessing

Convert title to lowercase

Count characters and words

Extract keywords

2. SEO SCORING

If keyword exists → + points

If keyword is at the beginning → extra points

If keyword density is natural → bonus

If missing keyword → penalty

Example:
seo_score = 0–100 based on:

keyword_presence (40%)

keyword_position (30%)

relevance (30%)

3. CTR SCORING

Check for:

Power words (e.g. “Ultimate”, “Secret”, “Proven”, “Shocking”)

Numbers

Brackets (e.g. [Guide], (2025))

Curiosity phrases

Example:
ctr_score = weighted score:

emotional_words

curiosity_gap

formatting

4. READABILITY SCORING

Sentence simplicity

Word length

Avoiding jargon

5. LENGTH SCORING

Ideal: 50–60 characters

Too short (<40) → penalty

Too long (>70) → penalty

6. FINAL SCORE

overall_score = average of:

SEO score

CTR score

readability score

length score
