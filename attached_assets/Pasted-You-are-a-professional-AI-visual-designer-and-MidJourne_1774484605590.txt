⚙️ CORE PROMPT GENERATION LOGIC
This is what ensures high-quality MidJourney prompts 👇

1️⃣ SUBJECT / CONCEPT ENGINE
Use user input for main concept

Augment with synonyms or descriptive keywords for clarity

Example:

Input: “Futuristic city”

Prompt: “Futuristic cyberpunk city skyline, neon lights, flying cars”

2️⃣ STYLE DESCRIPTOR ENGINE
Map style input to detailed descriptors

Examples:

Realistic → “photorealistic, ultra-detailed, 8k”

Illustration → “digital painting, smooth brush strokes”

Cartoon → “cartoon style, vibrant colors, expressive characters”

Cinematic → “cinematic lighting, dramatic shadows, wide-angle lens”

3️⃣ MOOD & LIGHTING ENGINE
Include lighting terms for realism and impact

Examples: “dramatic lighting, golden hour, volumetric light, soft shadows”

4️⃣ COLOR & COMPOSITION ENGINE
Add user-selected color palette (if provided)

Include descriptors for composition:

Minimal, cluttered, detailed, symmetrical, dynamic angles, portrait, wide shot, top-down

5️⃣ PERSPECTIVE / ANGLE ENGINE
Suggest angle based on visual type:

YouTube thumbnail → Close-up, dynamic perspective

Instagram post → Eye-level, centered composition

Course cover → Clean layout, minimalistic

Brand logo → Isometric, centered, minimal

6️⃣ ASPECT RATIO & QUALITY PARAMETERS
Default suggestions if user not provided:

Thumbnails: --ar 16:9

Instagram post: --ar 1:1

Stories / Reels: --ar 9:16

Quality: --q 2

Style: --style 4a or --v 6 (latest MidJourney engine version)

7️⃣ SYNTAX OPTIMIZATION
Combine descriptors with commas

Place parameters at the end: --ar 16:9 --q 2 --v 6

Avoid redundancy (don’t repeat words unnecessarily)

Ensure clarity and readability for MidJourney

8️⃣ EXPLANATION GENERATOR
For each prompt, add a 1–2 sentence explanation of the expected visual

Helps user visualize the output before generating

Example: “A cinematic cyberpunk cityscape at night with neon lights, flying cars, and glowing reflections.”

9️⃣ QUALITY CONTROL FILTER
Before returning prompts:

- Confirm prompt includes subject, style, lighting, composition, perspective, aspect ratio, quality params
- Ensure descriptors are specific, actionable, and visually inspiring
- Syntax is correct for MidJourney
- Ready for first-attempt generation
