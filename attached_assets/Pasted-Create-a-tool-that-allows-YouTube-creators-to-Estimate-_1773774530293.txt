⚙️ CALCULATION LOGIC
Definitions

CPM = Cost advertisers pay per 1000 monetized views

RPM = Revenue per 1000 total views

Step-by-Step Logic

Monetized Views:
monetized_views = total_views × (monetized_playback_percentage / 100)

If user provides CPM:
use it directly

If CPM is NOT provided:
estimate CPM based on:

Base CPM by Niche (example ranges)

Finance: $15–40

Tech: $10–25

Business: $12–30

Education: $8–20

Gaming: $3–10

Entertainment: $2–8

Lifestyle: $3–12

Health: $5–15

Adjust by Location

Tier 1 (US, UK, Canada, Australia): +40–100%

Europe: +20–50%

Global: baseline

India/Nigeria: -30–60%

Adjust by Engagement

High: +20–50%

Medium: baseline

Low: -20–40%

Adjust by Ad Types

More ad formats enabled = higher CPM multiplier

Revenue Formula

estimated_revenue = (monetized_views / 1000) × CPM

RPM Formula

RPM = estimated_revenue / total_views × 1000
