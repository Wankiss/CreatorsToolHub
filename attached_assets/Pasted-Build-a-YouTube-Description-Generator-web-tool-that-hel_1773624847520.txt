Core JavaScript Logic

Pseudo code:

onClick(generateButton)

    title = getTitle()
    topic = getTopic()
    keywords = getKeywords()
    timestamps = getTimestamps()
    cta = getCTA()

    intro = generateHook(title, keywords)

    body = generateParagraphs(topic, keywords)

    if timestamps exist
        formatTimestamps()

    hashtags = generateHashtags(keywords)

    description = combine(
        intro,
        body,
        timestamps,
        cta,
        links,
        hashtags
    )

    display(description)
