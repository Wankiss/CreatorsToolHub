Tag Generation Logic

The AI must generate tags using 5 layers of keyword expansion.

Step 1: Extract Core Keywords

From title + keywords:

Process:

• Convert to lowercase
• Remove stop words
• Extract primary nouns and phrases

Example:

Input title:

How to Grow on YouTube in 2026

Extract:

grow youtube
youtube growth
youtube tips
youtube strategy
youtube 2026

Create variations using:

Step 3: Generate Long-Tail Tags

Combine keywords into longer phrases.

Formula:

Primary Keyword + Modifier
Modifier + Primary Keyword

Simulate YouTube autocomplete patterns.

Add patterns like:

Tag Scoring System

Each generated tag should be scored using relevance.

Score formula:

Relevance Score =

+40 keyword match
+30 title similarity
+20 search intent match
+10 long-tail value

Sort tags by highest score first.

Tag Quality Rules

The AI must ensure:

✔ tags are 2-6 words long
✔ tags are human search phrases
✔ tags are not duplicates
✔ tags are not spammy

Avoid:

free free free
youtube youtube youtube
viral viral viral
