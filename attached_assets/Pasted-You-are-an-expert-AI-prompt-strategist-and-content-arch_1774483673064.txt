This is what makes your tool produce first-time success prompts 👇

1. ROLE-BASED ENGINE
Assign AI a clear persona for context

Examples:

You are a professional YouTube scriptwriter

You are a high-converting social media copywriter

You are an SEO-optimized blog writer

Logic:

If role not provided → assign default professional persona based on content type
2. CONTENT TYPE MAPPING
Content Type	AI Output Format	Notes
YouTube Script	Markdown, scene-by-scene, timestamps	Include hook, CTA
Instagram Caption	Short-form text, emoji optional	Include first line hook, CTA, hashtags
TikTok	Short script, POV, trend idea	Include first 3 seconds hook
Email Newsletter	Subject line, intro, body, CTA	Optional segmentation
Blog Post	Headings, subheadings, paragraphs	Optional SEO keywords
LinkedIn Post	Professional tone, concise	Include hook, insights, CTA
3. TONE & STYLE ENGINE
Map user-specified tone to AI instructions:

Formal → professional language, complete sentences

Casual → approachable, friendly

Humorous → playful, witty

Inspiring → motivational language

Bold → attention-grabbing, strong opinions

Logic:

Include explicit instruction: “Use [Tone] tone throughout the output”
4. LENGTH & STRUCTURE CONTROL
Set clear instructions:

Word count / character count

Paragraph / bullet / script format

Logic:

If not specified, assign default based on content type
5. REQUIRED COMPONENT ENGINE
Identify key elements per content type:

CTA, hashtags, bullet points, stats, storytelling

Logic:

Include bullet list of mandatory components in the prompt
6. STEP-BY-STEP INSTRUCTIONS
AI must know:

What role to take

What structure to use

Tone & style

Required components

Format for output

Logic:

Break into clear sequential instructions
7. EXAMPLE CONTENT (OPTIONAL)
Provide a small example snippet for clarity if requested

Logic:

Helps AI generate style-aligned content immediately
8. MULTI-PLATFORM ADAPTABILITY
Adjust instructions slightly depending on platform:

ChatGPT → conversational formatting

Claude → structured, precise instructions

Gemini → concise, professional output

Logic:

Include platform-specific tweaks in the structured prompt
9. QUALITY CONTROL FILTER
Before output:

- Prompt is clear, unambiguous
- Includes role, tone, format, components
- Ready for first-attempt success
- No missing information
- Copy-paste ready
